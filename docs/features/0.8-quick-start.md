# 0.8 快捷启动入口

## 目标

完善 Makefile 快捷命令，确保新人 clone 后 1 分钟能跑起来。

## 不做

- 不做 air 热重载安装（开发者自行 `go install`）
- 不做 CI/CD 集成

## 改造文件

| 文件 | 改动 |
|------|------|
| `Makefile` | 完善 quick-start 等待逻辑 + 新增 infra-status + 改进 help 输出 |

## 改动细节

1. `make infra-up` 增加健康检查等待（替代固定 sleep 5s）
2. `make quick-start` 改为串行执行（先确保 infra 健康再启动 run）
3. 新增 `make infra-status` — 查看基础设施容器状态
4. `make help` 输出更友好的中文说明

## 退出门

- [x] `make quick-start` 正常工作（中间件 Docker + 本地运行）
- [x] `make full-start` 正常工作（全 Docker）
- [x] `make help` 输出清晰的命令列表
- [x] `make infra-status` 正确显示容器状态
