# 0.7 Docker 应用镜像

## 目标

多阶段 Dockerfile 构建应用镜像，`make docker-up` 一条命令启动全栈（应用 + MySQL + PG + Redis + NATS），`/health` 全部 ok。

## 不做

- 不做生产 Nginx 反代 + 证书（阶段 12）
- 不做 CI/CD（阶段 12）
- 不做镜像推送到仓库

## 关键技术决策

### Go 1.25 Toolchain 兼容性

`go.mod` 声明 `go 1.25.0`，但 Docker Hub golang 官方镜像最高到 1.23。解决方案：
- 使用 `golang:1.23-alpine` 作为 builder 基础镜像
- 设置 `GOTOOLCHAIN=auto`，Go 1.23 会自动下载所需的 1.25 toolchain
- 或者改用 `golang:1.24-alpine`（如果已发布）

### Docker 环境配置文件

`APP_ENV=docker` 时加载 `configs/docker.yaml`：
- 数据库 DSN 指向容器服务名（`mysql:3306`、`postgres:5432`）
- Redis 地址为 `redis:6379`
- NATS URL 为 `nats://nats:4222`
- 日志格式改为 JSON（容器标准输出友好）
